# ТихийТыл — Инструкции для Claude

## Продукт

**Тихий Тыл** — приложение психологической поддержки для ветеранов боевых действий (Россия).
Сайт: https://тихийтыл.рф

Три модуля:
1. **Голосовой дневник** — запись голосом/текстом (ГОТОВ)
2. **Библиотека техник самопомощи** — персонализированная (В РАБОТЕ)
3. **Дашборд для психолога** — динамика клиента между сессиями (НЕ НАЧАТ)

## Принятые решения (не обсуждаются)

- ❌ Нет AI-диалогу — российские LLM цензурируют темы ПТСР/суицида, юридические риски
- ❌ Нет автоматическому анализу без участия психолога
- ✅ Голос и текст — равноценные варианты
- ✅ Психолог остаётся в контуре — не замена терапии

## Архитектура

- **Backend:** Flask 3.0 на порту **5002**, SQLite, Whisper (STT)
- **Frontend:** Vanilla JS, раздаётся Flask через `send_from_directory`
- **venv обязателен** — глобальная установка пакетов запрещена (см. VENV_INSTRUCTIONS.md)
- Запуск: `cd backend && source venv/bin/activate && python app.py`

## Стиль

- Название: **«Тихий Тыл»** (с пробелом)
- Шрифт: **Montserrat** (не Inter)
- Тёмная тема: `--bg-primary: #0F1923`, `--accent: #4A90A4`
- Типографика: заголовки `line-height: 0.9`, текст `line-height: 1.2`
- Внутренние отступы (padding) < внешних (margin/gap)

## Стиль кода

- Комментарии и документация — на русском
- Код (переменные, функции, классы) — на английском
- Не добавляй фичи сверх запрошенного
