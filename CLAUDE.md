# ТихийТыл — Product Context for Claude

## Что за продукт

**ТихийТыл** — приложение психологической поддержки для ветеранов боевых действий (Россия).

Три кита продукта:
1. **Голосовой дневник** — пользователь записывает голосом как себя чувствует
2. **Библиотека техник самопомощи** — персонализированная (показывает то, что работало раньше)
3. **Дашборд для психолога** — видит динамику клиента между сессиями

Сайт: https://тихийтыл.рф

## Принятые решения (не обсуждаются)

- ❌ **Нет AI-диалогу** — российские LLM цензурируют темы ПТСР/суицида, юридические риски
- ❌ **Нет автоматическому анализу** без участия психолога
- ✅ **Голосовой ввод** — основной формат, предпочтительнее текста
- ✅ **Психолог остаётся в контуре** — это не замена терапии, а инструмент между сессиями
