# ТихийТыл — Инструкции для Claude

## Продукт

**Тихий Тыл** — приложение психологической поддержки для ветеранов боевых действий (Россия).
Сайт: https://тихийтыл.рф

Модули:
1. **Голосовой дневник** — запись голосом/текстом (ГОТОВ)
2. **Библиотека техник самопомощи** — персонализированная (ГОТОВ)
3. **Профиль клиента** — код-позывной, статистика, настройки, рекомендации (ГОТОВ)
4. **Дашборд для психолога** — динамика клиента между сессиями (ГОТОВ)
5. **Домашние задания** — психолог назначает, клиент выполняет (ГОТОВ)

## Принятые решения (не обсуждаются)

- ❌ Нет AI-диалогу — российские LLM цензурируют темы ПТСР/суицида, юридические риски
- ❌ Нет автоматическому анализу без участия психолога
- ✅ Голос и текст — равноценные варианты
- ✅ Психолог остаётся в контуре — не замена терапии

## Архитектура

- **Backend:** Flask 3.0 на порту **5002**, SQLite, Whisper (STT)
- **Frontend:** Vanilla JS, раздаётся Flask через `send_from_directory`
- **Клиент:** `frontend/client/` — дневник, техники, профиль, домашки
- **Психолог:** `frontend/psychologist/` — отдельная папка, раздача через `/psychologist/`
- **Модули без Blueprint** — функции-импорты (diary.py, techniques.py, profile.py, psychologist.py, homework.py, cbt.py → app.py)
- **КБТ-сессии** — 5 обязательных вопросов при отказе от техники, переоценка боли, сохранение в БД, видно психологу (шторка раскрытия)
- **Дефолтные техники** — 1-я: Дыхание 4-7-8 (breathing, order=1), 2-я: Заземление 5-4-3-2-1 (grounding, order=1)
- **Пагинация дашборда** — дневник и КБТ по 10 записей, кнопка «Загрузить ещё»
- **Фильтр по дате** — клик по точке на графике или date-input фильтрует записи дневника
- **venv обязателен** — глобальная установка пакетов запрещена (см. VENV_INSTRUCTIONS.md)
- **Шкала боли** — медицинский стандарт: 1 = нет боли, 10 = невыносимая. Отрицательный pain_change = улучшение
- **Chart.js** (v4.4.0, CDN) — стандартная ось Y (снижение графика = улучшение)
- `request.get_json(silent=True)` вместо `request.json` (устойчивость к невалидному JSON)
- Запуск: `cd backend && source venv/bin/activate && python app.py`
