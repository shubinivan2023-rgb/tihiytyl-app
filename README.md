# 🤫 ТихийТыл — Голосовой дневник для ветеранов

> Приложение для психологической поддержки ветеранов с ПТСР

**Статус:** MVP Development  
**Версия:** 0.1.0

---

## 🎯 Что это

ТихийТыл — инструмент для ветеранов и психологов, работающих с ПТСР. Три модуля: голосовой дневник, библиотека техник самопомощи, дашборд психолога.

**Основные возможности:**

- **Дневник настроения** — записывай как прошёл день голосом или текстом (на выбор)
- **Техники самопомощи** — библиотека проверенных практик с голосовыми инструкциями
- **Статистика для психолога** — отслеживай динамику состояния клиента
- **Персонализация** — система запоминает какие техники работают лучше для конкретного человека

**Ключевые ограничения:**

• Никакого AI-диалога (цензура российских LLM + юридические риски)  
• Никакого автоанализа без психолога  
• Голос и текст — равноценные варианты (клиент выбирает удобный способ)

## 🚀 Быстрый старт

### Требования

- Python 3.9+
- pip
- SQLite3

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/shubinivan2023-rgb/tihiytyl-app.git
cd tihiytyl-app

# Установить зависимости
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Запустить сервер
python app.py
```

Открыть в браузере: `http://localhost:5003`

## 📦 Модули разработки

Проект разрабатывается итеративно по модулям:

- [ ] **Модуль 0:** Обновление лендинга
- [ ] **Модуль 1:** Базовый дневник (выбор режима: голос или текст)
- [ ] **Модуль 2:** Библиотека техник (категории + проигрыватель)
- [ ] **Модуль 3:** Профиль клиента (какие техники работают лучше)
- [ ] **Модуль 4:** Статистика (для клиента ограниченная, для психолога полная)
- [ ] **Модуль 5:** Доступ для психолога (код-позывной типа "БУРЯ-347")

См. `docs/modules/` для детальных ТЗ по каждому модулю
